Links

This particular page is about links. There are three kinds of links:

  • Absolute links, like to Mozilla Developer Network, which is a very good source for web developers.
  • Relative links, like to our extras page.

    • This page is about miscellaneous HTML things, but you may also be interested in links or images.

  • Root-relative links, like to the home page of our website, aren't useful to us right now because the files are local, not on a hosted web server.
